#####Bluemix Command#####

1. connect to IBM Bluemix

```
bluemix api https://api.ng.bluemix.net
```

2. log in to Bluemix

```
bluemix login -u username
```

3. push to Bluemix repository

```
cf push "app_name"
```

4. Access app in your browser

```
host.mybluemix.net
```
